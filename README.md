# Salary Prediction Using Linear Regression

## Overview

This project predicts employee salaries based on years of experience using a Linear Regression model. The model is trained on historical salary data and evaluated using RMSE and R² Score.

## Dataset

* YearsExperience
* Salary

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Workflow

1. Load the dataset
2. Split data into training and testing sets
3. Train the Linear Regression model
4. Predict salaries
5. Evaluate model performance
6. Save the trained model

## Results

The model successfully predicts employee salaries based on years of experience with good accuracy.
